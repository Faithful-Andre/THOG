# THOG Frontend

Frontend application for the **T.H.O.G (Total Health Operations Grid)** system.  
Built with modern web technologies to provide a fast, clean, and scalable clinical interface.

---

##  Tech Stack

- **Next.js 16 (App Router)**
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Zustand (state management)**

---

##  Features (Phase 1)

- Role-based dashboards (Clinician / Patient)
- Patient management UI
- Visit and vitals structure
- Sidebar navigation
- Clean responsive layout
- Cookie-based role handling (temporary)

---

##  Phase 2 (Planned)

- Real authentication (hashed passwords)
- Database integration
- JWT or session-based auth
- Real user roles from backend
- API-driven patient & visit data

---

npm run dev
